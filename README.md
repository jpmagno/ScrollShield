# ScrollShield

A mobile app that helps users detect, understand, and reduce doomscrolling behavior

## Tech Stack

- React Native + Expo
- SQLite (local data persistence)
- Zustand (state management)
- Victory Native (data visualization)

## Features (In Progress)

- [ ] Doomscroll session detection
- [ ] Behavioral pattern analysis
- [ ] Risk scoring system
- [ ] Gamification layer
- [ ] Personal insight reports
